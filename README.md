# シンプルなリズムゲーム・サンプル

これは、HTML, CSS, JavaScriptのみで作成された、リズムゲームの基本的な仕組みを学ぶためのサンプルプログラムです。

- **`falling-notes-game` フォルダ:** 上からノーツが降ってくるタイプ（ビートマニア、プロセカ風）
- **`side-scrolling-game` フォルダ:** 右からノーツが流れてくるタイプ（太鼓の達人風）

## 遊び方

1.  `index.html` ファイルをブラウザで開いてください。
2.  「ゲームスタート」ボタンを押してください。
3.  サンプル譜面が再生されます。

### 操作方法

**Falling Notes Game:**
- レーンに対応する `D`, `F`, `J`, `K` キーを押します。

**Side-Scrolling Game:**
- ドン（赤）: `F` または `J` キー
- カッ（青）: `D` または `K` キー
